# Employee Management System Bu proje Java ile geliştirilmiş basit bir çalışan yönetim sistemidir.
 Özellikler
- Çalışan ekleme (Create)
- Çalışan silme (Delete)
- Çalışan güncelleme (Update)
- Çalışan listeleme (Read)
- Katmanlı mimari (Service & Repository)
## Kullanılan Teknolojiler
- Java
- Spring Boot
- OOP (Object Oriented Programming)
##  roje Yapısı - EmployeeRepository.java - EmployeeServices.java - EmployeeServicesImpl.java 
## Amaç Bu proje Java OOP mantığını öğrenmek için yapılmıştır.
