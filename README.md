# Employee Management System
Bu proje, Java ve Spring Boot kullanılarak geliştirilmiş basit bir çalışan yönetim sistemidir. Temel CRUD (Create, Read, Update, Delete) işlemlerini gerçekleştirmek ve katmanlı mimari yapısını anlamak amacıyla oluşturulmuştur.

# Özellikler
- Çalışan ekleme (Create)
- Çalışan silme (Delete)
- Çalışan güncelleme (Update)
- Çalışan listeleme (Read)
- Katmanlı mimari (Service & Repository)
  
# Kullanılan Teknolojiler
- Java
- Spring Boot
- OOP (Object Oriented Programming)

# Proje Yapısı
-EmployeeRepository.java → Veri erişim katmanı
-EmployeeServices.java → Servis arayüzü
-EmployeeServicesImpl.java → Servis implementasyonu

# Amaç
Bu proje, Java OOP prensiplerini ve katmanlı mimari yapısını öğrenmek amacıyla geliştirilmiştir. Ayrıca backend geliştirme süreçlerine giriş niteliği taşımaktadır.
